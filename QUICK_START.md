# 🚀 Quick Start Commands

Copy and paste these commands in your terminal to get started:

## For Windows (Command Prompt or PowerShell):
```bash
cd path\to\hasaan-portfolio
npm install
npm run dev
```

## For Mac/Linux (Terminal):
```bash
cd path/to/hasaan-portfolio
npm install
npm run dev
```

---

## First Time Setup (After downloading the project):

### 1. Open Terminal/Command Prompt
- **Windows:** Press `Win + R`, type `cmd`, press Enter
- **Mac:** Press `Cmd + Space`, type `terminal`, press Enter  
- **Linux:** Press `Ctrl + Alt + T`

### 2. Navigate to project folder:
```bash
cd Desktop/hasaan-portfolio
```
*(Adjust path based on where you saved the folder)*

### 3. Install dependencies:
```bash
npm install
```
*Wait 1-3 minutes for installation to complete*

### 4. Start the development server:
```bash
npm run dev
```

### 5. Open your browser:
Go to: `http://localhost:3000`

---

## ✅ That's it! Your portfolio is now running!

The browser should open automatically. If not, manually go to `http://localhost:3000`

---

## 🛑 To Stop the Server:
Press `Ctrl + C` in the terminal

## 🔄 To Start Again:
```bash
npm run dev
```

---

## 📝 Common Commands:

| Command | What it does |
|---------|-------------|
| `npm install` | Install all project dependencies (only needed once) |
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |

---

## 🆘 If Something Goes Wrong:

### "npm: command not found"
→ Install Node.js from: https://nodejs.org/

### Port 3000 already in use
→ Close other apps or change port in `vite.config.js`

### Blank screen
→ Press F12, check Console tab for errors

### Changes not showing
→ Save file (Ctrl+S) and check terminal for errors
